# The Everyone App

> If you have a passion, welcome.

Most apps in the app store are built for one type of person — a gamer, a nature enthusiast, a social butterfly. They're all great. But what if one app had **everything**? Never focused on a single thing. Whoever you are, whatever you're into — there's something here for you.

**The Everyone App** is the one app for all of you.

---

## ✨ What's inside

A clickable HTML prototype showcasing the vision:

- **Drive** — Turn-by-turn directions with traffic, scenic routes, and EV stops
- **Fish** — Crowdsourced spots, species, and seasonal tips
- **Sky** — Real-time airspace, NOTAMs, and drone restrictions
- **Weather** — Beautiful, accurate, hyperlocal
- **Trails** — Hikes ranked by difficulty and view
- **Local** — Neighborhood spots people actually love

Plus a categorized "For everyone" section spanning twelve passion archetypes — Outdoors, Mind, Create, Move, Connect, Play, Home, Work, Wonder, Sound, Taste, and Care.

---

## 🚀 Getting started

This is a single-file static site. No build step. No dependencies.

### Run locally

```bash
git clone https://github.com/YOUR_USERNAME/everyone-app.git
cd everyone-app
open index.html
```

Or just double-click `index.html`.

### Deploy with GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under "Source," select `Deploy from a branch` → `main` → `/ (root)`
4. Save. Your site will be live at `https://YOUR_USERNAME.github.io/everyone-app/`

## 🎨 Design

- **Aesthetic:** Clean, modern, Apple-inspired — minimal but warm
- **Type:** Fraunces (display) + Geist (body)
- **Color:** Off-white canvas with restrained accent blue
- **Layout:** Generous whitespace, clear hierarchy, no clutter

The design choices serve the mission: an app for everyone shouldn't feel cluttered or overwhelming. It should feel like home.
